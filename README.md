# RHDS - Sistema de Recrutamento de Candidatos

## 📌 Sobre o Projeto
Sistema completo de gestão de vagas com painel empresa e controle de candidatos.

## 🚀 Tecnologias
- Java
- Spring Boot
- MySQL
- JWT
- MVC

## 🗄️ Banco de Dados
Modelagem relacional com as seguintes principais tabelas:

- alertas 
- candidato
- candidaturas
- empresa
- empresa_candidato
- etapa_processo
- experiencias
- formacoes
- pre_candidaturas
- processo_seletivo
- recrutador
- reset_senha
- unidade_empresa
- usuario
- vaga
- vaga_cnh
- vaga_formacao
- vaga_idioma
- vaga_localizacao
- vaga_requisitos

## 🔐 Autenticação
Sistema baseado em JWT com controle de roles (EMPRESA / CANDIDATO).
